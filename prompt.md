프론트엔드 개발자를 위한 포트폴리오 웹사이트를 제작해주세요.

## 기술 스택
- Vite + React
- Tailwind CSS v4
- Framer Motion

## 필수 기능
1. 완전한 반응형 디자인 (모바일, 태블릿, 데스크톱)
2. 다크모드/라이트모드 토글 기능
3. 부드러운 페이지 전환 애니메이션

## 디자인 방향
- 미니멀하고 깔끔한 UI
- 여백을 활용한 심플한 레이아웃
- 적절한 타이포그래피 계층 구조
- 화이트/다크 컬러 기반의 절제된 색상 팔레트

## 포함될 섹션
1. **Hero Section**: 이름, 직무, 간단한 소개, CTA 버튼
2. **About**: 자기소개, 기술 스택 (아이콘/뱃지 포함)
3. **Projects**: 프로젝트 카드 그리드 (이미지, 제목, 설명, 기술스택, 링크)
4. **Experience**: 경력/경험 타임라인
5. **Contact**: 이메일, GitHub, LinkedIn 등 연락처

## 애니메이션 요구사항 (Framer Motion)
- 스크롤 시 fade-in, slide-up 효과
- 호버 시 카드 lift 효과
- 페이지 전환 애니메이션
- 로딩 시 stagger 애니메이션
- 부드러운 다크모드 전환

## 추가 요청사항
- 한글 폰트 최적화 (예: Pretendard, SUIT)
- SEO 메타 태그 설정
- 스크롤 진행바 또는 내비게이션 인디케이터
- 프로젝트 필터링/카테고리 기능 (선택사항)

모든 텍스트는 한글로 작성하되, 더미 데이터로 채워주세요.
코드는 깔끔하게 컴포넌트 단위로 분리하고, 주석을 포함해주세요.
